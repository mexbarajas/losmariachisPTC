# Los Mariachis – Website

Authentic Mexican Restaurant · Peachtree City & Fairburn, GA

## Stack
- **React 18** + **Vite 5** 
- Cloudinary (auto-optimized images)
- GrubHub order integration

## Deploy to Vercel (2 minutes)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → Import repo
3. Vercel auto-detects Vite — no config needed
4. Click **Deploy** ✅

Build command: `npm run build`  
Output directory: `dist`  
Framework: **Vite** ← select this if asked

## Run Locally

```bash
npm install
npm run dev
```
